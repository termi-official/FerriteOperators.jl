# FerriteOperators.jl 

A library of common finite element operators with Ferrite.jl .

[![Stable](https://img.shields.io/badge/docs-stable-blue.svg)](https://termi-official.github.io/FerriteOperators.jl/stable/)
[![Dev](https://img.shields.io/badge/docs-dev-blue.svg)](https://termi-official.github.io/FerriteOperators.jl/dev/)
[![Build Status](https://github.com/termi-official/FerriteOperators.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/termi-official/FerriteOperators.jl/actions/workflows/CI.yml?query=branch%3Amain)
[![Coverage](https://codecov.io/gh/termi-official/FerriteOperators.jl/branch/main/graph/badge.svg)](https://codecov.io/gh/termi-official/FerriteOperators.jl)

This package serves as an intermediate layer between solvers and [Ferrite.jl]() modeling codes by providing a way to express operators in a unified way.
